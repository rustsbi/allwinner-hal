# Nezha D1-H examples

## UART example

```bash
cargo run -p nezha-d1-example --bin uart --target riscv64imac-unknown-none-elf --release
```

## Blinky example

```bash
cargo run -p nezha-d1-example --bin blinky --target riscv64imac-unknown-none-elf --release
```

## SD/MMC example

```bash
cargo run -p nezha-d1-example --bin sdmmc --target riscv64imac-unknown-none-elf --release
```
